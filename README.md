# 🚀 Jenson USA Sales Analysis using SQL

## 📌 Project Overview
This project analyzes the Jenson USA sales dataset using SQL to uncover meaningful insights about sales performance, customer behavior, and business operations.

The objective is to convert raw data into actionable insights that help in decision-making.

---

## 📊 Dataset
- Dataset provided by WsCube Tech
- Includes sales, customers, products, staff, and store data

---

## 🛠 Tools & Technologies
- SQL (MySQL)
- Data Cleaning
- Data Analysis

---

## 📋 Business Problems Solved

1. Total number of products sold by each store
2. Cumulative sales over time for each product
3. Highest revenue-generating product in each category
4. Customer who spent the most money
5. Highest-priced product in each category
6. Total orders by each customer per store
7. Staff members with no sales
8. Top 3 best-selling products
9. Median product price
10. Products never ordered
11. Staff performing above average
12. Customers who purchased from all categories

---

## 🧠 SQL Concepts Used
- Joins (INNER, LEFT)
- Aggregations (SUM, COUNT, AVG)
- GROUP BY & HAVING
- Subqueries
- Window Functions (RANK, ROW_NUMBER)
- CTEs
- EXISTS / NOT EXISTS

---

## 📈 Key Insights
- A small number of products generate most of the revenue
- Certain stores outperform others significantly
- Some staff members have zero or low sales
- High-value customers contribute heavily to revenue
- Some products are never purchased

---

## 💡 Business Recommendations
- Focus on high-performing products
- Improve marketing in low-performing stores
- Optimize inventory for unsold products
- Reward high-performing staff
- Target loyal customers with offers
